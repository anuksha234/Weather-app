# Weather-app
A responsive weather app built using HTML,CSS,and Javascript.It fetch and displays real-time weather data using the OpenWeatherMap API,including temperture,humidity,wind,speed and waether conditions.

## Features
- Search weather by city name
- Displays temperature
- Shows humidity and wind speed
- Responsive design
- Real-time weather information

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
